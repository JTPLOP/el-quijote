# EJERCICIO EVALUABLE GIT
Repositorio creado para realizar la práctica de clase sobre el uso básico de Git y GitHub en la asignatura de Entornos de Desarrollo.

## OBJETIVOS:

- `libros/quijote.txt`: contiene fragmentos de *El Quijote*.
- `README.md`: información del proyecto.
